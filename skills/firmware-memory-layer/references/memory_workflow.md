# Firmware Memory Workflow

1. Suggest memory from current artifacts.
2. Read only `read_now` suggestions.
3. Use memory as a lead and verify against current artifacts.
4. Save new reusable lessons as drafts.
5. Validate drafts before promotion.
6. Promote only safe, evidenced, reusable memory.

Drafts are not trusted memory. Promotion updates `memory-index.json`.
